# GitHub 2要素認証（2FA）の設定方法

GitHubで2要素認証（2FA）の設定が求められています。これはアカウントのセキュリティを高めるための設定です。

## 📱 設定方法（最も簡単）

### 方法1: 認証アプリを使う（推奨・簡単）

#### ステップ1: 認証アプリをインストール

スマートフォンに認証アプリをインストール：

- **iPhone**: App Storeで「Google Authenticator」を検索してインストール
- **Android**: Google Playで「Google Authenticator」を検索してインストール

（他の認証アプリでもOK：Microsoft Authenticator、Authyなど）

#### ステップ2: GitHubで2FAを有効化

1. GitHubの画面で「**Enable 2FA now**」ボタンをクリック
2. 「**Set up using an app**」を選択（推奨）
3. スマートフォンの認証アプリを開く
4. 「**+**」ボタンをタップして「**QRコードをスキャン**」を選択
5. GitHubの画面に表示されているQRコードをスキャン
6. 認証アプリに6桁のコードが表示されます
7. GitHubの画面にそのコードを入力
8. GitHubがバックアップコードを表示します → **必ず保存してください！**
9. 「**I have saved my recovery codes**」にチェックを入れて完了

**✅ これで2FA設定完了！**

---

### 方法2: SMS（電話番号）を使う

1. 「**Enable 2FA now**」ボタンをクリック
2. 「**Set up using SMS**」を選択
3. 国を選択（日本: +81）
4. 電話番号を入力（例: 09012345678）
5. 「**Send authentication code**」をクリック
6. スマートフォンに届いたコードを入力
7. バックアップコードを保存
8. 完了

---

## 🔑 バックアップコードについて

**重要**: バックアップコードは必ず保存してください！

- スマートフォンを紛失した場合
- 認証アプリを削除した場合
- スマートフォンが使えない場合

に、このコードでログインできます。

**保存方法**:
- テキストファイルに保存
- パスワード管理アプリに保存
- 印刷して安全な場所に保管

---

## 📝 今後の使い方

2FA設定後、GitHubにログインするときは：

1. ユーザー名とパスワードを入力
2. 認証アプリを開く
3. GitHub用の6桁のコードを確認
4. そのコードを入力

これでログイン完了です！

---

## ❓ よくある質問

### Q: 認証アプリのコードが表示されない
A: スマートフォンの時刻が正確か確認してください。時刻がずれているとコードが合いません。

### Q: バックアップコードを失くした
A: GitHubの設定ページから新しいバックアップコードを生成できます。

### Q: スマートフォンが使えない
A: バックアップコードを使うか、GitHubサポートに連絡してください。
