# リポジトリの場所を確認する方法

`omakase-receipt`フォルダが見つからない場合、GitHub Desktopでリポジトリの保存場所を確認します。

## 🔍 方法1: GitHub Desktopの設定で確認

1. **GitHub Desktopアプリを開く**
2. **メニューバーから「GitHub Desktop」→「Preferences」をクリック**
   - または、コマンド + カンマ（⌘ + ,）
3. **「Advanced」タブをクリック**
4. **「Default clone location」を確認**
   - ここに表示されているパスが、リポジトリの保存場所です
   - 例: `/Users/kant/Documents/GitHub`

---

## 🔍 方法2: GitHub Desktopのメニューから確認

1. **GitHub Desktopアプリを開く**
2. **「Repository」→「Show in Finder」をクリック**
   - リポジトリのフォルダがFinderで開きます
   - 開いたフォルダのパスを確認してください

---

## 🔍 方法3: GitHub Desktopのリポジトリリストから確認

1. **GitHub Desktopアプリを開く**
2. **左側のリポジトリリストで「omakase-receipt」を右クリック**
3. **「Show in Finder」を選択**
   - リポジトリのフォルダがFinderで開きます

---

## 📋 リポジトリが見つからない場合

### クローンが完了していない可能性

1. **GitHub Desktopで「omakase-receipt」リポジトリを選択**
2. **「File」→「Clone Repository」を再度実行**
3. **「URL」タブを選択**
4. **リポジトリのURLを入力**：
   ```
   https://github.com/tk1103/omakase-receipt.git
   ```
5. **「Local path」を確認・変更**
   - デフォルトの場所を確認
   - または、`/Users/kant/Documents/omakase-receipt` を指定
6. **「Clone」をクリック**

---

## 🎯 最も簡単な方法

GitHub Desktopのメニューから：
1. **「Repository」→「Show in Finder」をクリック**
2. **開いたフォルダのパスを確認**
3. **そのフォルダにファイルをコピー**

これで確実にリポジトリの場所がわかります！
